## Задание

Реализовать класс простых чисел с поддержкой следующих возможностей:

 - Указание диапазона
 - Поиск всех простых чисел в диапазоне 
 - Вывод чисел на экран

Все необходимые математические сведения можно найти в статье
["Простые числа"][prime] на сайте Wikipedia.

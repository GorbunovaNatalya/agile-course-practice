# Левосторонняя куча #
Выполнил:
- Герасимов Дмитрий
- ННГУ, ИИТММ, каф. МОСТ, группа 381603М-4

## Задание
Реализовать левостороннюю кучу с поддержкой следующих операций:
- Вставка элемента
- Извлечение минимума
- Удаление элемента
- Слияние двух куч

Всю необходимую информацию о левосторонней куче можно найти в статье ["Левосторонняя куча"][dheap].
